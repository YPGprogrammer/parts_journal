Приложение "Журнал запасных частей" является тестовым заданием, выполненным для ООО "Газпром "МКС".
Краткая инструкция для установки и запуска приложения:
Вам понадобится Docker и Docker Compose.
1. Скопируйте репозиторий командой git clone
2. Перейдите в скопированный репозиторий cd parts_journal
3. Запустите приложение через docker compose up -d --build
4. Переидите по адресу http://localhost:8501 для использования приложения
